Detailed Description

Main class of the memento pattern. It holds history of table states. Also it allows us to save state of table in some moment, or restore previous state of table

About it: we store there only previous states of originator (not current state).
We should to make backup() before any changes of originator. Call undo() to cancel last changing of originator.
